Система собирает информацию с датчиков, в ней есть модуль логирования, который заносит в журнал все обращения к датчикам.В системе
есть модуль, выполняющий обращения для получения данных с датчиков и модуль генерации htmlпредставления.
Запуск системы осуществляется из головного модуля.

Мы создали 5 модулей:
Модуль 1: сбор информации с датчиков
Модуль 2: логирование
Модуль 3: UI - работа с какимто предстовлением
Модуль 4: HTML-генератор
Модуль 5: главный модуль

Если более детально то по названию:
Модуль 1: data_provider
Модуль 2: logger
Модуль 3: user_interface
Модуль 4: html_creater
Модуль 5: main 

Далее в рамках каждого из этих модулей мы опишем некоторый набор методов:
data_provider - метод получает:
get_temperature (получить температуру), get_pressure (получить давление), get_wind_speed (получить скорость ветра)

logger - метод регистрирует:
temperature_logger (регистратор температуры), pressure_logger (регистратор давление), wind_speed_logger (регистратор скорость ветра)

user_interface - метод получает:
temperature_view (температурный режим), pressure_view (измерения давления), wind_speed_view ( измирения ветра)

html_creater - метод плучает:
create - создать приложения если надо сохранить, передать.....
main - метод получает:
в которой будет одна кнопка запустить систему
